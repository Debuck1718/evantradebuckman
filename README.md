# Evantra De-Buckman Ventures — Corporate Website

A modern, professional, and responsive website for Evantra De-Buckman Ventures (Ghana). Built as a lightweight static site (HTML/CSS/JS) suitable for deployment on Netlify, GitHub Pages, Vercel, or any static host.

## Structure
- Home: index.html
- About: about.html
- Services: services.html
- Industries: industries.html
- Why Choose Us: why-us.html
- Contact: contact.html
- Assets: /assets/css, /assets/js, /assets/images

## Local Development
Use any static server:
- VS Code Live Server extension
- Python: `python3 -m http.server 8080` (serve from project root)
- Node: `npx serve .`

Open http://localhost:8080

## Customize
- Update company details and contact in `/assets/js/site.js` (siteConfig).
- Replace placeholder images in `/assets/images`.
- Update copy within page sections as needed.

## SEO
- Edit meta tags in each HTML head: title, description, and Open Graph tags.
- Provide a favicon at `/assets/images/favicon.png`.
